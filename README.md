# V-Memo (バレーボール得点板オーバーレイ・エディタ)

V-Memo（ブイ・メモ）は、バレーボールの試合映像に対して、美しくカスタマイズ可能な得点板（スコアボード）のオーバーレイを合成・編集・エディットできるデスクトップアプリケーションです。
Tauri (Rust) と React (TypeScript) をベースに構築されており、軽量かつ高速に動作します。

---

## 主な機能

### 1. 試合のイベント履歴・スコアリング
- **スコア連動の入力システム**: ポイント追加、サーブ権の交代、セットの確定や取り消し（Undo）をタイムスタンプ付きで記録します。
- **タイムラインとスコア推移グラフ**: 動画プレイヤーの下部でスコアの推移をビジュアルグラフ（時系列）で視覚化し、試合の流れを俯瞰できます。

### 2. 多彩な得点板デザイン & カスタマイズ
- **4種のテーマ**:
  - 🎨 **グラスモーフィズム** (半透明の磨りガラス風)
  - 🎨 **ミニマル・フラット** (シンプル・モダン)
  - 🎨 **レトロ・デジタル** (7セグメントデジタル時計風)
  - 🎨 **テレビ中継風バー** (プロのテレビ放送風)
- **テーマカラー**: チームごとのカラー（デフォルトは白）をカラーピッカーから自由に変更できます。
- **配置・サイズ変更**: 得点板の大きさ（%）や画面上の四隅（左上、右上、左下、右下）の位置設定に対応し、動画の大きさに連動してスケールします。

### 3. 音声自動変換 (MP3 ➔ AAC 高速互換変換)
- **無音バグの自動解消**: ブラウザエンジン (WebKit) で再生できない「MP3音声付きMP4動画」を読み込んだ際に自動的に検知します。
- **高速エンコード**: 映像は一切劣化させずにコピーし、音声トラックのみを互換性の高い `AAC` 形式に変換（`ffmpeg` 使用）。長い試合動画でもわずか数秒で変換が完了し、アプリ内で即座に音が鳴る動画としてロードされます。

### 4. 動画切り出し ＆ エクスポート
- **プリセット登録**: 特定のシーン（セット間、ラリーなど）をイン点・アウト点で登録し、プリセットとして保存可能。
- **合成エクスポート**: 映像・音声フェードや、番組表風のタイトルカード（日付・イベント名・対戦カード）を自動で挟み込み、高画質な動画（または透過素材用のProRes WebM）を書き出すことができます。

### 5. アプリ全体のテーマ切り替え
- **3つのカラーテーマ**: 
  - 🌙 **モダン・ダーク** (デフォルト・サイバー感のある暗色)
  - ☀️ **クリーン・ライト** (見やすい明るい白ベース)
  - 🕶️ **ミッドナイト・ブラック** (AMOLED有機EL向け漆黒)

## 技術スタック

* **フロントエンド**: React 18, TypeScript, Vite
* **デスクトップランタイム**: Tauri v2
* **バックエンド**: Rust
* **メディアエンジン**: FFmpeg / FFprobe (システムまたはHomebrew)

## 開発と実行の手順

### 前提条件
システムに `ffmpeg` および `ffprobe` がインストールされており、環境変数 `PATH` が通っている必要があります。

#### macOS の場合
Homebrew で簡単にインストール可能です：
```bash
brew install ffmpeg
```

#### Windows の場合
以下のいずれかの方法でインストールし、コマンドプロンプトや PowerShell から `ffmpeg` コマンドが実行可能になるように設定（環境変数 PATH への追加）を行ってください。

1. **winget（Windows パッケージマネージャー）を使う場合 (推奨)**:
   管理者権限の PowerShell で以下を実行します：
   ```powershell
   winget install Gyan.FFmpeg
   ```
2. **Scoop を使う場合**:
   ```powershell
   scoop install ffmpeg
   ```
3. **手動でインストールする場合**:
   [Gyan.dev](https://www.gyan.dev/ffmpeg/builds/) 等からビルド済みの zip ファイルをダウンロードし、任意のフォルダに解凍後、その中の `bin` フォルダのパスをシステムの環境変数 `PATH` に追加します。

---

### セットアップ
リポジトリをクローンしたのち、依存パッケージをインストールします：
```bash
npm install
```

※Windows 環境で Tauri アプリをコンパイル（開発・ビルド）するには、**C++ Build Tools (Visual Studio Build Tools)** のインストールが必要です。

---

### 開発モードの起動
Tauri の開発環境を立ち上げます：
```bash
npm run tauri dev
```

---

### プロダクションビルド
アプリパッケージを作成します：
```bash
npm run tauri build
```
実行完了後、OSに適合したインストーラーおよびパッケージファイルが自動生成されます：
- **macOS**: `src-tauri/target/release/bundle/dmg/` 以下に `.app` および `.dmg` が生成されます。
- **Windows**: `src-tauri/target/release/bundle/msi/` 以下にインストーラー（`.msi`）が生成されます。
